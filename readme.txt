Extends the JSON API Plugin


==Description==


JSON API User is a plugin that extends the JSON API Plugin with a new Controller to allow RESTful create blog mu

Features include:

* Create blog mu

==Installation==

First you have to install the JSON API for WordPress Plugin (http://wordpress.org/extend/plugins/json-api/installation/).

To install JSON API User just follow these steps:

* upload the folder "json-api-mu" to your WordPress plugin folder (/wp-content/plugins)

* activate the plugin through the 'Plugins' menu in WordPress or by using the link provided by the plugin installer

* activate the controller through the JSON API menu found in the WordPress admin center (Settings -> JSON API)
